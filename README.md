# RTP

> The RTP exists to preserve engineering knowledge.

---

## What is RTP?

The **Research Transmission Protocol (RTP)** is an open-source infrastructure designed to preserve engineering knowledge.

Rather than allowing important engineering decisions, investigations and discoveries to disappear over time, RTP provides a structured way to preserve and recover them.

Knowledge should outlive implementations.

---

## Why?

Engineering projects naturally produce knowledge.

Architectural decisions.

Investigations.

Experiments.

Lessons.

Most of this knowledge is eventually lost.

RTP exists to ensure it is preserved.

---

## Foundation

The first implementation of RTP is built around four pillars:

- Knowledge preservation
- Engineering documentation
- Structured investigations
- A Linux laboratory server (**rockyd**)

---

## Current Status

**Phase I — Foundation**

The project is currently focused on building its core infrastructure.
